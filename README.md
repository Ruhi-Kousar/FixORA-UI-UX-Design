# 🚀 FixORA - Report • Resolve • Rebuild

## 📌 Overview

FixORA is a mobile application UI/UX design created in Figma that helps citizens report public issues, volunteer for community activities, and monitor issue resolution.

The application aims to improve communication between citizens, volunteers, and authorities through an intuitive and user-friendly interface.

---

## ✨ Features

- User Login & Registration
- Issue Reporting
- Upload Photos
- Live Map View
- Volunteer Registration
- Community Events
- Notifications
- Leaderboard
- User Profile
- Settings
